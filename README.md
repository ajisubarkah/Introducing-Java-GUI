# Hai Gaes

Hai gaes ini repository penggunaan frame di Java. Berikut fitur yang ditawarkan :

- Table auto refresh
- Combo box dinamis
- Data mahasiswa dinamis
- Form tambah mahasiswa di frame beda
- Form mahasiswa memiliki return value
